**Other valuable wedges:**
- **Real NFL schedule formula** ŌĆö attempted in this session but deferred.
  The pairing generator (division round-robin + 3-year intra rotation +
  4-year inter rotation + same-place finishers) is straightforward; the
  hard part is **week assignment** of 272 specific game pairings into 17
  weeks such that each team plays exactly once per week. Greedy / MRV
  heuristics get stuck on the regular structure of the NFL multigraph
  (32 nodes, 17-regular). Picking this up later needs a real edge-coloring
  algorithm (Misra-Gries Vizing, or constructive 1-factorization using
  division-week / non-division-week scheduling blocks). Current `schedule.ts`
  uses random perfect matchings ŌĆö works robustly, just doesn't match the
  NFL pairing rules.
- **Roster Manager UI** ŌĆö depth chart drag-and-drop, scheme-fit indicators
